このプロジェクトについて
------------------------
完全に独自設計のカスタムCSSで構築されたウェブサイトです。
Tailwind CSSを一切使用せず、100%カスタムCSSで設計しています。

ファイル構成
------------
juntendo-custom-css/
├── index.html          メインHTMLファイル（735行）
├── css/
│   └── style.css      完全カスタムCSS（900行以上、Tailwind不使用）
├── js/
│   ├── main.js        jQuery実装のメインスクリプト
│   └── data.js        店舗データ
└── README.txt         このファイル

特徴
----
✅ 完全カスタムCSS - Tailwindを一切使用していません
✅ セマンティックなクラス名
✅ 読みやすく保守しやすいCSS設計
✅ レスポンシブデザイン対応
✅ jQuery実装による動的機能
✅ Swiper.jsによるスライダー
✅ 美しいグラデーションとアニメーション

セクション構成
--------------
1. ヘッダー（ナビゲーション）
2. メインビジュアル（デュアルスライダー）
3. お知らせ（NEWS）- フィルター機能付き
4. 商品情報（FAIR）
5. 店舗情報（SHOP INFO）- 都道府県切り替え機能
6. ポイントカード（POINT CARD）
7. 関連リンク（LINKS）
8. 法人向けセクション
9. 会社概要（COMPANY）
10. フッター

使い方
------
1. index.htmlをブラウザで開く
2. または、以下のコマンドでローカルサーバーを起動：
   
   # Python 3の場合
   python3 -m http.server 8084
   
   # Python 2の場合
   python -m SimpleHTTPServer 8084
   
   その後、http://localhost:8084 にアクセス

動的機能
--------
• 上部・下部スライダー（Swiper.js使用）
• モバイルメニューの開閉
• ニュースフィルター
• 都道府県別店舗表示切り替え
• マップ画像の切り替え
• トップへ戻るボタン

CSS設計の特徴
--------------
• BEM風のクラス命名規則
• セクションごとに明確に分類
• カスケードとスペシフィシティを適切に管理
• メディアクエリによるレスポンシブ対応
• CSS変数不使用（シンプルな設計）
• グリッドとフレックスボックスを効果的に使用

カスタマイズ方法
----------------
1. 色の変更：css/style.css の各セクションのカラーコードを変更
2. レイアウトの変更：グリッドやフレックスボックスのプロパティを調整
3. 店舗データの変更：js/data.js を編集
